SPB documentation: deployment

Production

  • The production version is Kimball (172.22.32.4).
  • The application is installed and runs with user 'spb'
  • Updating the code is done by running 'git pull' in the ~/spb directory.
    This will update both code and documentation
  • The nginx configuration is stored in /etc/nginx/conf.d/groundcontrol.0x20.be.conf
